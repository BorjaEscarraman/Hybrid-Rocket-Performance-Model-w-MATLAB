# Hybrid-Rocket-Performance-Model

Started from Zero to learn where to direct myself when building a MATLAB model out of class setting. Realizing that just because you have done a lot of math, doesn't mean you are able to create any math model you want. It takes a while to wrap your head around the problem, and take that physical representation of it and make a mathematical model you can juggle around with to approach simulations and enhanced dedicate sub-parts of the system.
